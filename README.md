## Homeworks

This section contains the homework assignments completed during the course. Each homework focuses on applying specific concepts discussed in lectures, such as system modeling or software design. The tasks typically involve implementing small practical examples, analyzing given problems, or producing diagrams and documentation related to software engineering topics.

## Presentations

This section contains in class presentations created during the semester:

* **Short presentations (5 minutes)** – brief presentations where students introduce a selected topic, tool, or concept related to the course. The goal is to quickly explain the core idea and demonstrate understanding of the topic.
